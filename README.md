# CAMP

A visual interface for viewing crimes throughout NYC.

## Project Structure

```
repo/
├─ backend/ # Python backend
├─ frontend/ # React frontend
```

## Setup

### 1. Backend
Create a virtual environment then navigate to `backend` folder:

```bash

python -m venv venv
source venv/bin/activate #mac
venv\Scripts\activate #windows with command prompt

pip install -r requirements.txt #install dependencies 

cd backend #navigate to backend



python app.py #run backend

```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
